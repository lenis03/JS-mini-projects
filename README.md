# JavaScript Mini Projects

A collection of small JavaScript projects created for learning and practice.

## Projects

- Rock Paper Scissors
- Calculator
- To-Do List
- More projects coming soon...

## Technologies Used

- HTML
- CSS
- JavaScript
- Local Storage API

## Purpose

This repository serves as a place to practice JavaScript concepts and improve coding skills by building small, self-contained projects.

Each project focuses on different topics such as:

- DOM Manipulation
- Event Handling
- Functions
- Arrays and Objects
- Local Storage
- Timers
- Keyboard Events

## Running the Projects

Clone the repository and open any project's `project-name.html` file in your browser.

```bash
git clone https://github.com/lenis03/JS-mini-projects.git
```

## Notes

These projects are primarily educational and may be refactored or improved over time.
